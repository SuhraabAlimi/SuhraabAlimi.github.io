Insertion sort with the reversed text is the next longest function, followed by Selection Sort Ordered, Reverse, and Random, which were all very close to each other, at about 175,000 ms for 100,000 words. Insertion Sort reversed is in this position because it would have to switch for every new word added. Insertion sort random is the next slowest, at about 117,000 ms.
